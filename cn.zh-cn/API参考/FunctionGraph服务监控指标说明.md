# FunctionGraph服务监控指标说明<a name="ZH-CN_TOPIC_0134295092"></a>

## 功能说明<a name="section37797504358"></a>

本节定义了FunctionGraph服务中“函数”上报云监控服务的监控指标。

指标的命名空间和维度，您可以通过云监控服务提供的API接口来检索FunctionGraph服务中“函数”产生的监控指标和告警信息。
